\titul{Programovací jazyk Rust jako náhrada za jazyk C++}
{Adam Hrnčárek}{Mgr. Marek Blaha}{Brno 2023}

\podekovani{Chtěl bych poděkovat Rust komunitě za skvělé učební materiály,
    díky kterým Rustu rozumím. Také bych chtěl poděkovat YouTuberovi TheCherno,
    jehož C++ série mě naučila všechny koncepty uvedené v této práci.
    Na závěr bych rád poděkoval svému profesoru informatiky a vedoucímu práce
    Marku Blahovi.
}

\prohlasenimuz{V~Brně dne \today}

\abstract{This thesis investigates the pros and cons of the Rust programming
    language compared to C++. This thesis tries to focus more on larger projects,
    as the differences are more apparent.
}{Specifically, this thesis focuses on the safety features of Rust, developer tooling,
    (testing, language servers, documentation, package management) and productivity
    with them. It also looks at reliability of these languages in the future.
}
\abstrakt{Tato závěrečná práce posuzuje výhody a nevýhody programovacího jazyka Rust
    v~porovnání s~C++. Práce se snaží soustředit spíše na větší projekty, při kterých
    se rozdíly mezi jazyky projeví více.
}{Specificky se práce soustředí na bezpečnostní prvky Rustu, nástroje pro vývojáře
    (testování, language servery, dokumentace, package management) a produktivitu
    s~nimi. Také se zabírá spolehlivostí těchto jazyků do~budoucna.
}

\klslova{Programovací Jazyk Rust, C++, porovnání jazyků, vývojářské nástroje,
    vývojářská produktivita, softwarová spolehlivost a bezpečnost
}
\keywords{The Rust Programming Language, C++, language comparison, developer tooling,
    developer productivity, software reliability and safety
}
